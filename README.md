# github-ossinsight
一个 Tampermonkey(油猴) 脚本，它可以帮助您在 GitHub 仓库中快速链接到 [ossinsight](https://ossinsight.io/) ， 以便快速分析开源项目健康状况以更在多维度深入探究一个开源项目

## 安装

- Greasy Fork 安装: [点击这里](https://greasyfork.org/scripts/456878-github-ossinsight)
